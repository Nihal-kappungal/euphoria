# Euphoria - Fashion Store Website

A modern and responsive fashion e-commerce landing page built using HTML, CSS, and JavaScript.

## ✨ Features

- Responsive design
- Fashion product showcase
- Men & Women categories
- New arrivals section
- Brand deals section
- Customer testimonials
- Clean and modern UI

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

```bash

```

3. Launch `index.html` in your browser
